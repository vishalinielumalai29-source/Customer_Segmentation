# 🛒 Mall Customer Segmentation with K-Means

This repository demonstrates **customer segmentation** using **K-Means clustering** on the Mall Customer dataset. Customers are grouped based on **Annual Income** and **Spending Score**, providing actionable insights for **marketing strategies and customer engagement**.

## Key Features
- `Annual Income (k$)` – Customer income  
- `Spending Score (1-100)` – Customer spending behavior  

## Workflow
1. Load and preprocess the dataset  
2. Scale features using `StandardScaler`  
3. Determine optimal clusters (Elbow Method)  
4. Apply K-Means clustering  
5. Visualize clusters with scatter plots  
6. Save segmented data for analysis  

## Tools
- Python 3, Jupyter Notebook  
- Libraries: `pandas`, `scikit-learn`, `matplotlib`, `seaborn`  

## Repository Contents
- `Mall_Customers.csv` – Original dataset  
- `Customer_Segmentation.ipynb` – Notebook with clustering workflow  
- `Mall_Customers_Segmented.csv` – Dataset with cluster labels  

---

**Outcome:** Clear segmentation of customers into actionable groups, helping businesses target marketing effectively.
